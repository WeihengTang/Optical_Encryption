Abstract

We propose MetaLens: a framework that leverages a physically-fabricated metasurface lens as an optical encryption device. The metasurface imposes a complex, spatially-varying point-spread function (SV-PSF) — with a distinctive ring/donut shape and off-axis aberrations — on every captured scene.

An authorized receiver who possesses the PSF (the secret key) applies patch-wise non-blind deconvolution and reconstructs the scene. An adversary who attempts blind deblurring — even with state-of-the-art algorithms — cannot recover meaningful content, because the true PSF is spatially varying, ring-shaped, and unknown.

Our long-term goal is to optimize the metasurface parameters end-to-end to maximize the authorized vs. adversary performance gap, using a differentiable PSF generator trained jointly against both sides.

# Encryption
scene → [metasurface lens] → encrypted image y

# Authorized (knows PSF θ)
y + θ → Non-blind Wiener → x̂  ✓ recognizable

# Adversary (no PSF knowledge)
y      → Blind deblurring → x̃  ✗ noise

The Metasurface PSF

The PSF is stored as a (7, 7, 101, 101) array — a 7 × 7 spatial grid of kernels, each 101 × 101 pixels. The ring structure rotates and gains coma-like asymmetry towards the field corners, making blind estimation from a single blurred image extremely difficult.

Method

🔑 Authorized — Non-blind Wiener

Patch-wise Wiener deconvolution with known SV-PSF. Each of the 49 patches is independently inverted via:

X̂(ω) = H*(ω) / (|H(ω)|² + λ) · Y(ω)

λ = 5 × 10⁻² regularisation. Kernel is zero-shifted to (0,0) before FFT to prevent phase offset.

🚧 Adversary — Blind Wiener-Hunt

scikit-image's Unsupervised Wiener (Gibbs sampler, Gaussian image prior) applied to the whole image, assuming a single spatially-uniform blur. The estimator converges to a smeared average kernel — not the true spatially-varying ring PSF.

Optimization Framework (Proposed)

Once the differentiable metasurface PSF generator is integrated, we will jointly optimize the PSF parameters φ to maximize the security gap end-to-end:

Phase 1: train both decoders with PSF fixed. Phase 2: freeze decoders, back-propagate ∂ΔL/∂φ to update metasurface params.
