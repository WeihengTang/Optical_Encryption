Abstract

We propose MetaLens: a framework that uses a physically-fabricated metasurface lens as an optical encryption device. The metasurface imposes a spatially-varying point-spread function (SV-PSF) — featuring a characteristic ring/donut shape with off-axis aberrations — on every captured scene, producing an image that appears as structured noise to an observer without the key.

An authorized receiver who possesses the PSF (the secret key) applies patch-wise non-blind deconvolution and faithfully reconstructs the original scene. An adversary who attempts blind deblurring — even with state-of-the-art algorithms — cannot recover meaningful content, because the true PSF is spatially-varying, ring-shaped, and unknown.

Our long-term goal is to optimize the metasurface parameters end-to-end to maximize the authorized vs. adversary performance gap, using a differentiable PSF generator jointly trained against both decoders.

The Metasurface PSF

The metasurface produces a spatially-varying PSF stored as a (7 × 7 × 101 × 101) array — 49 distinct kernels arranged on a spatial grid, each 101 × 101 pixels. The kernels exhibit a ring/donut structure that rotates and develops asymmetric coma-like tails towards the field corners, a hallmark of off-axis meta-optic aberrations.

This complex, position-dependent shape is precisely what makes blind PSF estimation from a single blurred image computationally intractable. A blind deblurring algorithm that assumes a spatially-uniform kernel will converge to a smeared average — far from the true local kernel at any position.

Method

🔑 Authorized Decoder

The authorized receiver applies patch-wise Wiener deconvolution with the known SV-PSF:

X̂(ω) = H*(ω) / (|H(ω)|² + λ) · Y(ω)

Each of the 49 image patches is independently deconvolved using its corresponding kernel (71 × 71 crop, 96.6% energy), with λ = 5 × 10−2 regularisation. The kernel is zero-shifted to position (0,0) before DFT to eliminate phase offsets.

🚧 Blind Adversary

The adversary applies scikit-image's Unsupervised Wiener-Hunt (a Gibbs sampler under a Gaussian image prior) to the entire blurred image, with no knowledge of the PSF.

The algorithm assumes a single spatially-uniform blur and initialises with a small Gaussian kernel. Because the true PSF is spatially-varying and ring-shaped, the estimator converges to a wrong, smeared kernel — producing a result that is visually unrecognisable.

Optimization Framework (proposed)

Once the co-worker's differentiable PSF generator is integrated, we will run an adversarial training loop to find a metasurface parameter set φ that maximises the security gap end-to-end:

maxφ  PSNR(Dnb(Bφ(x), φ), x) − λ · PSNR(Db(Bφ(x)), x)
s.t. φ ∈ physically realizable metasurface parameter space

Roadmap
