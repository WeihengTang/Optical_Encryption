# MetaLens — Optical Encryption via Spatially-Varying Metasurface PSF

> **Can a metasurface lens encrypt a scene so that only someone with the PSF can read it?**

This project explores using a physically-fabricated **metasurface** as an optical encryption device.  The lens imposes a spatially-varying, ring-shaped PSF on any captured scene.  An *authorized receiver* who knows the PSF applies non-blind deconvolution and recovers the image.  An *adversary* who relies on blind deblurring cannot.

---

## Concept

```
Scene  →  [Metasurface lens]  →  Encrypted image y

Authorized:   y + PSF  →  Non-blind Wiener  →  x̂   ✓ recognizable
Adversary:    y        →  Blind deblurring  →  x̃   ✗ unrecognizable
```

Long-term goal: **optimize the metasurface PSF parameters** to maximise the performance gap between authorized and blind recovery, using a differentiable PSF generator (in progress).

---

## Current PSF

`SV_PSF.npy` — shape `(7, 7, 101, 101)`: a 7 × 7 grid of spatially-distinct PSF kernels, each 101 × 101 pixels, generated by a physical metasurface design.  Kernels exhibit a characteristic ring/donut structure with off-axis aberrations at field corners.

## Preliminary Results (classical baselines, no training required)

| Image     | Encrypted | Non-blind (auth.) | Blind (adv.) | Gap ↑  |
|-----------|-----------|-------------------|--------------|--------|
| camera    | 16.3 dB   | 14.1 dB           | 4.8 dB       | **+9.3 dB** |
| astronaut | 15.6 dB   | 14.4 dB           | 4.7 dB       | **+9.6 dB** |
| chelsea   | 18.6 dB   | 16.0 dB           | 5.8 dB       | **+10.2 dB** |

*Non-blind*: patch-wise Wiener deconvolution with known SV-PSF.
*Blind*: Unsupervised Wiener-Hunt (Gibbs sampler) — treats blur as spatially uniform.

## Proposed Optimization Framework

```
Phase 1 — Train decoders (PSF fixed)
  Loss = −PSNR(D_nb(B_θ(x), θ), x) + λ·PSNR(D_b(B_θ(x)), x)
  Update D_nb and D_b.

Phase 2 — Optimize PSF (decoders frozen)
  max_φ  PSNR(D_nb(B_φ(x), φ), x) − λ·PSNR(D_b(B_φ(x)), x)
  subject to φ ∈ physically realizable metasurface parameter space
  Update φ via ∂ΔL/∂φ (differentiable PSF generator).

Repeat until convergence.
```
